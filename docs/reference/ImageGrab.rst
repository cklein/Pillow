:mod:`ImageGrab` Module (Windows-only)
======================================

The :py:mod:`ImageGrab` module can be used to copy the contents of the screen
or the clipboard to a PIL image memory.

.. note:: The current version works on Windows only.

.. versionadded:: 1.1.3
